# Finance Dashboard UI

## Overview

This project is a frontend Finance Dashboard built using React and Tailwind CSS.
It allows users to view financial summaries, explore transactions, analyze spending patterns, and simulate role-based access.

The project uses mock data and focuses on clean UI, component structure, and state management.

---

## Features

* Dashboard Overview
* Total Balance, Income, Expenses
* Monthly Financial Trend Chart
* Spending Category Pie Chart
* Transactions Table
* Search and Filter Transactions
* Sort by Date and Amount
* Role-Based UI (Viewer/Admin)
* Add, Edit, Delete Transactions
* Insights Section
* Responsive Design
* Local Storage Support

---

## Installation

Clone the repository

git clone <your-repo-link>

cd finance-dashboard

Install dependencies

npm install

Run project

npm run dev

---

## Role-Based UI

Viewer:
Can view dashboard and transactions

Admin:
Can add, edit, and delete transactions

Role can be switched using dropdown.

---

## Assumptions

* Mock financial data is used
* No backend required
* Role-based UI is simulated
* Data stored in local storage

---

Project can be deployed using Vercel or Netlify.
